# Expense Tracker

A modern web application for tracking personal expenses, built using React and Appwrite for authentication and storage.


## Features

- User registration and authentication (Appwrite)
- Add, edit, and delete expenses
- Categorize expenses (Food, Transport, Entertainment, Other)
- View expenses by category, date, and total amount
- Filter expenses by category and month
- Responsive and user-friendly design
